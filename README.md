# Arduino DHT11 Temperature Monitor

## Features
- DHT11 temperature sensing
- Software debouncing
- Start/Stop system using push button
- Non-blocking timing using millis()
- LED status indication

## Components Used
- Arduino Uno
- DHT11 Sensor
- Push Button
- Red LED
- Green LED
- 220Ω Resistors
